Images

Images can be displayed in HTML, in HTML, images are defined with the <img> tag.

The <img> tag is empty, it contains attributes only, and does not have a closing tag.

The src attribute specifies the URL (web address) of the image:

<img src="url" alt="some_text" style="width:width;height:height;">

For example:

<img src="https://goo.gl/nVydIR" alt="HTML5 Icon" style="width:128px;height:128px;">

Bulleted Lists

Lists can be displayed in HTML There are two types of lists, orderd and unordered.

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.

The list items will be marked with bullets (small black circles) by default:

<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>

An ordered list starts with the <ol> tag and each list item starts with the <li> tag as before.

<ol>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

This is what will be displayed:

An Unordered List

  • Coffee
  • Tea
  • Milk

An Ordered List

  1. Coffee
  2. Tea
  3. Milk

Tables

Tables can also be displayed in HTML.

An HTML table is defined with the <table> tag.

Each table row is defined with the <tr> tag. A table header is defined with the <th> tag. By default, table headings are bold and centered. A table data/cell is defined with the <td> tag.

For example:

<table style="width:100%">
  <tr>
    <th>Firstname</th>
    <th>Lastname</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>50</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>94</td>
  </tr>
  <tr>
    <td>John</td>
    <td>Doe</td>
    <td>80</td>
  </tr>
</table>

Will display:

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94
John Doe 80
